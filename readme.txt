Este proyecto es un template con boilerplate muy común para juegos, específicamente pensado para
empezar rápido en jams. Sigues necesitando ser programador y tener familiaridad con Godot para usarlo correctamente.

Examina el header de los scripts de /Globals para ver para lo que sirven y en lo que te pueden ayudar.
Modifica el menú de /MainMenu a tu gusto.
Acuérdate de introducir todos los textos en /Locale/locale.csv usando algún editor externo (recomiendo VSCode).
Cambia el /Themes/standard_ui_theme a tu gusto (o no).
Añade /UI/PauseMenu/pause_menu.tscn a cualquier escena invisible y el menú de pausa funcionará automáticamente.
Añade /Misc/volume_control_asp.gd a los reproductores de audio que uses para que tengan en cuenta el volumen de opciones.
Puedes abrir el menú de opciones para cambiar rápidamente las opciones mientras desarrollas. Está integrado en el menú principal y el de pausa de cara al jugador, pero se puede integrar en otros sitios.
